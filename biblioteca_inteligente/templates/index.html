Biblioteca Inteligente

Sistema online de gerenciamento de bibliotecas escolares.

Sobre nós

A Biblioteca Inteligente é um software de gerenciamento com objetivo de registrar movimentações de livros numa biblioteca. Através deste, os usuários poderão se cadastrar no sistema, ver disponibilidade de livros e ter controle sobre as próprias movimentações.

A evolução tecnológica permitiu que processos técnicos dentro das instituições educacionais, seja em relação aos estudos ou ao gerenciamento interno, fossem adaptados tecnologicamente para métodos mais precisos e rápidos. Apesar disso, não é possível acompanhar esse avanço dentro de grande parte das escolas públicas. Neste projeto, propomos uma solução para esse desfalque dentro das bibliotecas públicas, por meio da criação de um sistema de software de fácil acesso e uso.
